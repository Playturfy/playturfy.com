Super easy! 1) Start your tournament match on mobile, 2) Tap the red cast icon (organizers only), 3) Copy the broadcast link, 4) Open OBS on your PC, 5) Add Browser Source and paste the link, 6) Set size to 1920x1080, 7) Done! Your overlay updates in real-time as you record match events on mobile.
Works with any software that supports browser sources: OBS Studio (free, recommended), Streamlabs Desktop, XSplit Broadcaster, vMix, Wirecast, Restream Studio, StreamYard, Lightstream, Prism Live Studio, and more! If it can display a web page, it can display your overlay.
Only tournament organizers (the person who created the tournament) can access broadcasting controls. This ensures secure access and prevents unauthorized streaming. If you created the tournament, you'll see a red cast icon on match cards - tap it to get your broadcast link!
