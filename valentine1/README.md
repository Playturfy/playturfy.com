## Setup

1. **Start with the question page**: Open `question.html` in a modern web browser
2. **Interactive Question**: 
   - Click "Yes" to see the congratulations and automatically redirect to the main celebration page
   - Try clicking "No" - the button will run away from you! (It changes position every time)
   - The "Yes" button grows bigger each time you try to click "No"
3. **Main Celebration**: After clicking "Yes", you'll be redirected to `index.html` with the full Valentine's experience
4. **Background Music**: Add a file named `romantic-music.mp3` in the same directory for music
5. Enjoy the fun romantic experience!
